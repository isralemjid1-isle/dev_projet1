# jeu de mot

Ce projet est un mini-jeu où le programme choisit automatiquement un mot aléatoire depuis un fichier JSON, et l’utilisateur doit le deviner en un maximum de 6 essais
# Fonctionnement du jeu

- Le programme pioche un mot au hasard dans un fichier JSON.
- L’utilisateur dispose de 6 tentatives pour deviner le mot.
- Après chaque essai :
  - lettre correcte et bien placée:affichée en vert  
  - lettre correcte mais mal placée :affichée en jaune  
  - lettre absente du mot affichée en rouge  
- les lettres correctement trouvées sont automatiquement recopiées dans la ligne suivante.

# Indices disponibles

Le joueur dispose de 3 indices, utilisables une seule fois chacun :
1.La première lettre du mot  
2. La dernière lettre du mot  
3. La longueur du mot  

Ces indices ne peuvent être consultés qu’une seule fois.
#Structure des données
Les mots possibles sont stockés dans un fichier JSON, depuis lequel le programme choisit un mot aléatoire.
